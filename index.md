# Latent Extraction Systems

I build research systems that extract hidden structure from complex behaviour, then use that structure for model repair, uncertainty estimation, forecasting, and decision-making.

Across my work, I treat modelling as latent extraction:

> complex system → high-dimensional observations → latent regions/regimes → targeted intervention → measured behaviour change

## Flagship work

### [MorpheusDB](./morpheusdb.html)

A latent behavioural analysis system for model and system behaviour. MorpheusDB audits predictions, extracts behavioural features, clusters failures into regions, supports drilldowns, and generates targeted DPO/GRPO repair curricula with regression anchors.

**Recent result:** using a Morpheus-selected DPO curriculum, strict exact accuracy improved from **79.10% to 79.83%** over 689 overlapping audited rows, with one behavioural region improving from **66.67% to 72.22%**.
